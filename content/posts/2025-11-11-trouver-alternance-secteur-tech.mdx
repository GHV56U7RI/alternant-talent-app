## Le secteur tech : un eldorado pour l'alternance

Le secteur technologique offre des milliers d'opportunités en alternance chaque année. Startups, scale-ups, grands groupes : tous recherchent des talents pour renforcer leurs équipes.

## Les profils recherchés

### Développement
- Développeur web (front, back, fullstack)
- Développeur mobile (iOS, Android, React Native)
- DevOps / SRE

### Data & IA
- Data analyst
- Data scientist
- Machine Learning Engineer

### Produit & Design
- Product manager
- UX/UI Designer
- Product designer

## Où chercher ?

### Plateformes spécialisées
- **Mon alternance talent** (évidemment !)
- Welcome to the Jungle
- Chooseyourboss
- LinkedIn

### Réseaux d'écoles
Ne négligez pas le réseau alumni de votre école : c'est souvent la meilleure source d'opportunités.

### Salons et événements
- Forum de l'alternance
- Meetups tech locaux
- Hackathons

## Optimiser son profil

### GitHub / Portfolio
Montrez vos projets personnels. Un profil GitHub actif vaut mieux qu'un CV parfait.

### LinkedIn
- Photo professionnelle
- Résumé accrocheur
- Projets mis en avant
- Recommandations

## La candidature parfaite

### Email de motivation
- Court (max 10 lignes)
- Personnalisé
- Montrez que vous connaissez l'entreprise
- Mettez en avant vos projets

### CV tech
- 1 page maximum
- Stack technique visible
- Projets avec liens GitHub
- Réalisations chiffrées

## Préparer l'entretien technique

### Tests techniques
Entraînez-vous sur :
- Codewars
- LeetCode
- HackerRank

### Questions fréquentes
- Expliquez un projet complexe
- Comment gérez-vous les bugs ?
- Quelles technologies vous intéressent ?

## Négocier sa rémunération

Le salaire en alternance tech varie selon :
- Votre niveau (Bac+3, Bac+5)
- L'entreprise (startup vs grand groupe)
- La localisation (Paris vs province)

**Fourchettes indicatives :**
- Bac+3 : 1000-1300€
- Bac+5 : 1400-1800€

N'hésitez pas à négocier, surtout si vous avez des compétences recherchées !
