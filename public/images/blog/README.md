# Blog Images

Ce dossier contient les illustrations pour les articles de blog.

## Organisation

- Placez vos images ici avec des noms descriptifs
- Format recommandé : PNG ou JPG
- Taille recommandée : 1200x630px pour les couvertures d'articles

## Utilisation

Dans vos articles MDX, référencez les images comme ceci :
```
cover: '/images/blog/mon-image.jpg'
```
